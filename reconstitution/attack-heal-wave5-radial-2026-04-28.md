## Claim Attacked

The finite-`N` radial-parts theorem asserts, conditionally on the
Harish-Chandra--Wallach--Levasseur--Stafford input, that

```tex
\operatorname{Rad}_0(J_N(f))
  =
  \Delta^{-1}\left(\sum_{i=1}^N
    \operatorname{Op}_{\mathrm W}
    (f)(\lambda_i,-\hbar\partial_{\lambda_i})\right)\Delta
```

and therefore

```tex
D_N(f,g)=
\hbar^{-1}[J_N(f),J_N(g)]-J_N(\{f,g\}_\hbar)
```

vanishes in degree-zero quantum Hamiltonian reduction.

## Strongest Failure Mode

The cited primary lineage supports the Harish-Chandra homomorphism,
regular-Cartan landing after discriminant conjugation, and
Levasseur--Stafford kernel/surjectivity context. It does not, as presently
anchored, state the manuscript's exact Weyl-trace image formula with

```text
Y^i_j = -hbar d/dX^j_i,
z2 = -hbar d/dlambda,
J_N(f) = Tr Op_W(f)(X,Y).
```

Thus the theorem is not source-closed at all `N`. The exact Weyl-trace
image normalization must remain a named conditional input.

## Local Computation

Added a direct rank-two falsification harness to
`scripts/check_moyal_coefficients.py`. It realizes
`\mathcal W_2 = D_\hbar(\mathfrak{gl}_2)` on coordinates

```text
X = [[x00,x01],[x10,x11]],  Y^i_j = -hbar d/dX^j_i.
```

For invariant test functions generated by `Tr(X^r)`, it computes the
actual matrix Weyl differential operator `J_2(f)`, restricts to diagonal
matrices, and compares with `Delta^{-1}S_2(f)Delta`.

Checked operators:

```text
z2, z2^2, z1 z2, z1^2 z2, z1 z2^2, z1^2 z2^2.
```

Checked invariant test functions:

```text
1, Tr X, Tr X^2, Tr X^3, (Tr X)^2, (Tr X)(Tr X^2).
```

Result: all 36 exact rational comparisons pass with symbolic powers of
`\hbar`.

Concrete constant check:

```tex
J_N(z_1z_2)=T_{1,1}-\frac{\hbar N^2}{2}.
```

For `N=2`, the Cartan operator gives

```tex
\Delta^{-1}S_2(z_1z_2)\Delta
  =-\hbar(\lambda_1\partial_{\lambda_1}
          +\lambda_2\partial_{\lambda_2})-2\hbar,
```

matching the matrix Euler operator restricted to invariant functions.

## Exact Constants Retained

Moyal raw commutator:

```tex
[f,g]_{\mathrm{raw}}
 =\sum_{r\geq1,\ r\ {\rm odd}}
   \frac{2}{2^r r!}\hbar^r P^r(f,g).
```

First and third raw coefficients:

```tex
\hbar P(f,g),\qquad \frac{\hbar^3}{24}P^3(f,g).
```

Capelli/Weyl triangular system:

```tex
J_N(z_1^a z_2^b)
 \equiv
 \sum_{r=0}^{\min(a,b)}
   \left(-\frac{\hbar N}{2}\right)^r
   r!\binom ar\binom br\,T_{a-r,b-r}.
```

Normal-ordering defect:

```tex
\hbar^{-1}[T_{2,1},T_{0,2}]
  =4T_{1,2}-2\hbar N\,T_{0,1}.
```
