## Attack-Heal Ledger

### A246-01: Costello graph/QME source overreach

Claim attacked: Costello or Costello--Li might already prove the
Hamiltonian brane-defect non-scalar graph/QME theorem.

Finding: attack succeeds.  The Costello source fixtures support only the
universal finite-scale elliptic BV/RG/QME calculus.  They do not support a
current-valued target, a `D'_c` wavefront/counterterm theorem, or a
bulk-to-defect kernel.

Heal: keep Costello as source for heat kernels, regularized BV Laplacian,
RG/QME, local counterterms, graph asymptotics, and obstruction grammar.
Require internal construction for the Hamiltonian current target, scalar lift,
finite-row table, theta3 row, counterterms, and non-scalar QME kernel.

Status: source boundary valid.

### A246-02: LQT citation gap

Claim attacked: the LQT surface has theorem-grade primary support in the
local source bundle.

Finding: attack succeeds.  `main.tex` has correct bibliography entries for
Loday--Quillen 1984 and Tsygan 1983, but `references/primary-sources/` has no
local LQT/Tsygan primary fixture.  Costello--Li contains secondary/thematic
LQT references, not a replacement for primary anchors.

Heal: add a source fixture for LQ84/Tsy83 with exact theorem/page anchors.
Do not use LQT to prove the manuscript's Dirac/Koszul trace calculation.

Status: missing source fixture.

### A246-03: Tate bar-cobar/Quillen source gap

Claim attacked: the admissible Tate bar-cobar/Quillen envelope is source
closed by the bibliography.

Finding: attack succeeds.  The bibliography has Priddy, Loday--Vallette,
Hinich, Hovey, Lefevre-Hasegawa, Lurie, Quillen, and Schwede--Shipley, but no
local fixture with exact theorem anchors.

Heal: source the standard bar-cobar/model-category package precisely, then
prove the admissible Tate envelope, lower-central pronilpotence,
continuous-dual exactness, Roos/Milnor conditions, and strict endpoint
obstruction internally.

Status: missing source fixture plus internal theorem obligation.

### A246-04: CE/PV source dependence

Claim attacked: finite-window CE/PV requires an external source theorem to be
valid.

Finding: attack fails.  The finite-coordinate CE/PV map is internally
computed and can stand without an external theorem.  External sources would
only fix standard Schouten/PV conventions if the prose wants them.

Heal: keep the finite CE/PV identity as internal.  Treat completions,
equivariant refined brackets, kernel admissibility, and global descent as
separate internal theorem surfaces.

Status: internal proof, no blocking citation gap.

### A246-05: Matlis principal-part support

Claim attacked: the Matlis principal-part surface may be missing source
support.

Finding: attack fails for the top-local-cohomology source row.  The Matlis /
Hartshorne / Kunz fixture supplies publisher-grade anchors for
`E_R(k)`, the Cech-Laurent basis, and the residue pairing.

Heal: cite the fixture for the local cohomology/residue basis.  Keep the
reduced non-`R`-submodule statement, Hamiltonian coadjoint formula, residue
rigidity, and PBW/Matlis no-go as internal computations.

Status: source support closed, internal algebra remains manuscript proof.

### A246-06: Wavefront/Hormander citation gap

Claim attacked: wavefront-admissible current graph calculus is already
source supported.

Finding: attack succeeds.  `hormander-vol1` is in the bibliography, but no
local source fixture gives exact wavefront product/pullback/pushforward
anchors.  More importantly, Hormander would only supply criteria, not the
manuscript's kernel wavefront computations.

Heal: add a Hormander Ch. 8 source fixture and derive the graphwise
wavefront, diagonal-extension, counterterm, interval/truncation, and
arbitrary-current obstruction theorem internally.

Status: missing source fixture and internal microlocal theorem obligation.

### A246-07: Weiss/Ran descent source gap

Claim attacked: Costello--Gwilliam metadata or AFT analogy proves the global
Weiss/Ran descent surface.

Finding: attack succeeds.  The Costello--Gwilliam local HTML mirrors are
metadata/vocabulary support only.  The AFT/omega fixture records useful
theorem grammar for stratified factorization homology but is not integrated
as a manuscript citation package and does not construct the mixed HT
coefficients.

Heal: add theorem-grade CG/AFT/Weiss/Ran anchors as needed.  Prove the
product-disk mixed site, stratified pair `(X,L)`, collar modules, centrality
homotopies, descent vector, and trace state internally.

Status: missing source package and internal descent obligation.

### A246-08: BMK source gap

Claim attacked: the Bochner--Martinelli--Koppelman finite-window transfer is
source closed.

Finding: partial attack succeeds.  The manuscript supplies an internal
finite-window proof, but there is no primary source anchor for the classical
BMK identity and normalization.

Heal: add a several-complex-variables primary source for the BMK/Koppelman
identity.  Keep cutoff collar estimates, finite moment projection, arity-two
transfer, and all-window obstruction vector internal.

Status: missing classical source anchor; local transfer remains internal.

### A246-09: Weyl/Moyal/radial source boundary

Claim attacked: radial-parts sources prove the exact Weyl-trace formula used
by the manuscript.

Finding: attack succeeds.  The radial source fixture explicitly found no
primary source for the exact bivariate triangular trace formula or exact
all-`N` Weyl-trace radial normalization with the manuscript's sign/order
conventions.

Heal: cite Harish-Chandra/Wallach/Levasseur--Stafford only for the radial
component framework.  Prove the exact triangular formula, radial
normalization, edge PBW telescope, and interior decorated-necklace theorem
internally, or retain the all-`N` normalization as an explicit external input.

Status: framework sourced; exact formula internal/conditional.

### A246-10: Omega and physical trace analogies

Claim attacked: Costello Omega, Costello M2/Koszul, AFT, or CFG prove the
manuscript's normal Omega and stratified trace theorem.

Finding: attack succeeds.  The source fixture records analogies and theorem
grammar only.  The local brane-preserving normal scaling torus, localized
coefficient ring, residue/Euler normalization, QME quantization, and
physical large-`N` trace state are not supplied externally.

Heal: if used in the manuscript, add exact bibliography/source anchors for
AFT, Costello Omega, Costello M2/Koszul, and CFG in their own scopes.  Build
the local Omega/trace theorem internally.

Status: analogy only; internal construction required.
