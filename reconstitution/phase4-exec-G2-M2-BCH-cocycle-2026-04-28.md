## §1. T1 — Identify the degree-3 generators in the BCH series (M2.1)

### §1.1 BCH series, depth-by-depth

For Hamiltonians H_1, H_2 ∈ bar A = C[z_1, z_2] / C·1, the BCH
formula is:

```
log(exp(H_1) exp(H_2)) =  H_1 + H_2                     -- depth 1
                       +  (1/2) {H_1, H_2}              -- depth 2
                       +  (1/12) ({H_1, {H_1, H_2}}
                                   + {H_2, {H_2, H_1}}) -- depth 3
                       -  (1/24) {H_2, {H_1, {H_1, H_2}}}-- depth 4
                       +  ...                            -- depth ≥ 5.
```

The **depth-3 (cubic) coefficient** is

```
BCH_3(H_1, H_2) = (1/12) ({H_1, {H_1, H_2}} + {H_2, {H_2, H_1}}).
```

This is symmetric under (H_1 ↔ H_2) by direct computation — verified
in script T6 by re-symmetrization of ω_3_alt.

### §1.2 Cubic-degree generators on the 3-dim solvable Borel

The Phase-4 G2 milestone P4-G2-02 reduced cocycle compatibility to
the **9 Hamiltonian generators at degrees 1-3**:

```
   degree 1:  z_1, z_2.
   degree 2:  z_1²,  z_1 z_2,  z_2².
   degree 3:  z_1³,  z_1² z_2,  z_1 z_2²,  z_2³.
```

Of these, the **3-dim solvable Borel** of W3-W26-T1 is generated by
⟨z_1, z_2, z_1 z_2⟩. The cubic terms of BCH for inputs ranging over

- **{H_1, {H_1, H_2}}** with H_1 = z_1, H_2 = z_2:
  inner = {z_1, z_2} = 1 ∈ C ↦ 0 mod constants in bar A;
  outer = {z_1, 0} = 0. **Trivial**.

- **{H_1, {H_1, H_2}}** with H_1 = z_1², H_2 = z_2:
  inner = {z_1², z_2} = -2 z_1; outer = {z_1², -2 z_1} = -2 · {z_1², z_1}
  = -2 · 0 (since {z_1², z_1} = 2·1·z_1 - 0 = 0 by Poisson on

  inner = {z_1 z_2, z_1} = -z_1; outer = {z_1 z_2, -z_1} = z_1.
  **Non-zero** but lands in a degree-1 Hamiltonian, NOT a constant.

- **{H_1, {H_1, H_2}}** with H_1 = z_1, H_2 = z_1 z_2:
  inner = {z_1, z_1 z_2} = z_1; outer = {z_1, z_1} = 0. **Trivial**.

### §2.2 The structural verdict

By the Jacobi identity in the unreduced Poisson algebra
(`mathfrak h_poly` = C[z_1, z_2] before mod-constants),

```
   {h_1, {h_2, h_3}} + {h_2, {h_3, h_1}} + {h_3, {h_1, h_2}} = 0.
```

The constant projection of this sum is zero, so

```
   ω_3_unreduced(h_1, h_2, h_3) + ω_3_unreduced(h_2, h_3, h_1)
                                + ω_3_unreduced(h_3, h_1, h_2) = 0,
```

which is the **cyclic-sum vanishing**. The alternating ω_3_alt
inherits this cyclic property and additionally vanishes under the
3-cycle anti-symmetrization (the alternating-sum is identically 0
because the sum is already cyclically zero).

**Therefore ω_3_alt = 0 as a 3-cochain on bar A.** The cocycle
condition d_CE ω_3_alt = d_CE 0 = 0 holds **trivially**.
