## Stable Core

The finite-window graph-array schema now has an order-two boundary-composition
entry.  For distinct first scalar-contact faces \(\beta_i,\beta_j\), the
composite incidence coefficient is
\[
  \iota_{\beta_j|\beta_i,\beta_i}
    =
  {(-1)}^{i-1}{(-1)}^{\operatorname{pos}_i(j)-1}
  \varepsilon_{\beta_i}\varepsilon_{\beta_j|\beta_i}.
\]
The two-contact scalar gate is recorded as
\[
  S^{(2)}_{\beta_j|\beta_i,M}
    =
  \iota_{\beta_j|\beta_i,\beta_i}
  \left(
    \prod_{L}
    \operatorname{Str}_{\mathrm{cyc}}(\mathfrak m'_L)
  \right)
  \omega(f_{\beta_i},g_{\beta_i})
  \omega(f_{\beta_j|\beta_i},g_{\beta_j|\beta_i})
  \gamma_{\mathbf 1}^{(2)}.
\]
Here \(\gamma_{\mathbf 1}^{(2)}\) is only the ordered two-contact
associated-graded symbol; no target-algebra \(\gamma_{\mathbf 1}^2\)
survival is asserted.

For the minimal full-equivariant closed array generated by the strict
current-interface rows and the one-cross scalar-contact row, with no genuine
order-two seed graph, no order-two CE source face, no order-two counterterm
face, and \(C_{1,M}=0\), the order-two residual is zero:
\[
  R^{\mathrm{ns}}_{2,M}=0,\qquad C_{2,M}=0.
\]
The proof uses two facts: codimension-two boundary composites cancel with
opposite incidence signs, and every full-equivariant cyclic marker trace on
\(\C^{N|N}\) is zero, including the empty marker word.

For any larger package the order-two residual remains an explicit row sum,
not a computed number:
\[
\begin{aligned}
  R^{\mathrm{ns}}_{2,M}
   ={}&
    \sum_{|\Gamma|_\hbar=2}
      \varepsilon_\Gamma d_M K^M_\Gamma
    -\sum_{|\Gamma'|_\hbar=2}
      \varepsilon^{\mathrm{CE}}_{\Gamma'}K^M_{\Gamma'}\\
    &+\frac12
    \sum_{|\Gamma_1|_\hbar=|\Gamma_2|_\hbar=1}
      \varepsilon_{\Gamma_1,\Gamma_2}
      \{K^M_{\Gamma_1},K^M_{\Gamma_2}\}_{\hbar,M}\\
    &+[\hbar^2]\left(
      d_M C_{<2,M}
      +\frac12\{C_{<2,M},C_{<2,M}\}_{\hbar,M}\right).
\end{aligned}
\]
