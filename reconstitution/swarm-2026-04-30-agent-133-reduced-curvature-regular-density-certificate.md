## Stable Core

The certificate closes for the actual residual obtained by evaluating the
regular-density finite-window graph kernel:

```tex
\operatorname{Ob}^{\mathrm{red,reg}}_{w,\partial,\hbar,M}
  =
\operatorname{ev}_{\mathcal G_M}
  \operatorname{Curv}_{\mathbf K}
  (\kappa^{\mathrm{graph,reg},M}_{\hbar,w,I}).
```

If all cotangent defect labels are \(b(t)dt\in\mathcal D_c^{\mathrm{reg}}(I)\),
the scalar projection is zero on the chosen subhabitat, and lower-order
counterterms lie compatibly in
\(\mathcal Q^{0,\mathrm{rd}}_{w,M}(I)=
\mathcal X^{0,\mathrm{reg}}_{w,M}(I)\), then

```tex
\mathfrak o^{\mathrm{ns,reg}}_{n_0,M}
  =
  [\hbar^{n_0}]\left(
    \operatorname{Ob}^{\mathrm{red,reg}}_{w,\partial,\hbar,M}
    +d_MC^{\mathrm{reg}}_{<n_0,M}
    +\frac12\{C^{\mathrm{reg}}_{<n_0,M},
              C^{\mathrm{reg}}_{<n_0,M}\}_{\hbar,M}
  \right)
```

lies in
\[
Z^1(\mathcal Q^{\bullet,\mathrm{rd}}_{w,M}(I))
=Z^1(\mathcal X^{\bullet,\mathrm{reg}}_{w,M}(I)).
\]

The explicit representative is

```tex
W_{n_0,M}=-\mathfrak o^{\mathrm{ns,reg}}_{n_0,M},
\qquad
C_{n_0,M}=0.
```

Thus
\[
\eta^{\mathrm{reg}}_{n_0}
=\kappa^{\mathrm{reg}}_{n_0}
=\beta^{\mathrm{cl},\mathrm{reg}}_{n_0}
=\mu^{\mathrm{cl}}_{n_0}
=\lambda^{\mathrm{cl}}_{n_0}=0
\]
on the regular-density branch.

If the actual finite-window package contains a wavefront-admissible but
non-smooth cotangent label, or a distributional counterterm forced by such
a label, that datum leaves \(Q^{\mathrm{rd}}=X^{\mathrm{reg}}\).  The
quotient component is

```tex

  =
\left[
q_M[\hbar^{n_0}]\left(
  \operatorname{Ob}^{\mathrm{red},\neg\mathrm{rd}}_{w,\partial,\hbar,M}
  +d_MC^{\neg\mathrm{rd}}_{<n_0,M}
  +\{C^{\mathrm{reg}}_{<n_0,M},
      C^{\neg\mathrm{rd}}_{<n_0,M}\}_{\hbar,M}
  +\frac12
    \{C^{\neg\mathrm{rd}}_{<n_0,M},
       C^{\neg\mathrm{rd}}_{<n_0,M}\}_{\hbar,M}
\right)
\right].
```
