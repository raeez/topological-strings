## Verdict

The corrected finite-window habitat survives the attack.

The dg window must be

```text
A_{psi,W}=A_psi/F^{>W}A_psi
```

where `F^{>W}` is the two-sided dg ideal generated by words of total
Koszul weight `>W` for

```text
wt(x)=(1,0), wt(y)=(0,1), wt(psi)=(1,1).
```

This is a genuine finite-dimensional unital dg quotient.  Ordinary
word-length windows are false: `d psi=xy-yx` raises ordinary word length.

The LQT map is valid only in the defined single-cycle habitat:

```text
Prim^{1cyc}_{N,W}:=im Theta_{N,W},
lambda_{LQT,W}=Theta_{N,W}^{-1}
```

and, on all stable invariant CE chains, only after constructing the
chain projection `P_{1cyc}`:

```text
lambda_{LQT,W}=Theta_{N,W}^{-1} P_{1cyc}.
```

The scalar quotient is not valid if it kills `Theta_1(psi)` alone:

```text
d_CE Theta_1(psi)=Theta_1(xy-yx).
```

It becomes a chain quotient only by killing the two-term scalar-Koszul
subcomplex on source and target, or after passing to homology:

```text
Theta_1(psi) -> Theta_1(xy-yx),
[psi] -> [xy-yx].
```

The current TeX records this correctly.  No manuscript edit is needed
from this report.

   Correct: those sources support the stable homological template only.

   separate dg-LQT source.
2. False: `Prim C_*^CE(gl_N(A_{psi,W}))` is a literal finite-N chain
   domain.  Correct: the domain is the stable single-cycle image
   `im Theta_{N,W}`, or a stable invariant complex after applying a
   constructed `P_{1cyc}`.
3. False: ordinary letter-length cutoffs define a finite-window dg
   algebra.  Correct: only Koszul-weight quotient windows are dg-stable.
4. False: removing the scalar one-`psi` line is a chain quotient.
   Correct: the differential target `Theta_1(xy-yx)` must be killed in
   the same subcomplex, or the quotient must be delayed to homology.
