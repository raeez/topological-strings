closed-exchange class
\[

\[

## Construction Attempt

The attempted algebraic repair is the cone
\[
  \mathcal X_{\mathrm{alg},M}^{1}=\mathbb C[[\hbar]]e_M,\qquad d^X_Me_M=0,
\]
\[
  \Xi^{\mathrm{alg},\mathrm{sc}}_M(e_M)
  =
  -\hbar N\,\bar c_M(f,g)
  \int_Ia(t)b(t)\gamma_{\mathbf 1}(t)\,dt.
\]
With \(W_M=e_M\), \(C_M=0\), and \(\rho^X_{M,N}e_M=e_N\), it gives zero
fixed-window and Roos obstructions inside the enlarged central-contact
branch. This is not a Costello-local closed exchange because it directly
adjoins the contact value and supplies no graph/Hom construction.

The only non-tautological central-source alternative is
\[
  \bar A^{\mathrm{cl}}_{-\hbar N\bar c}
  =
  \bar A\oplus\mathbb C K_{\mathrm{cl}},
  \qquad
  [(f,aK),(g,bK)]
  =
  (\{f,g\}_{\bar A},-\hbar N\bar c(f,g)K).
\]
For the central dual \(k^\vee\),
\[
  d_{\mathrm{CE}}(-k^\vee)=-\hbar N\bar c.
\]
Thus the desired opposite scalar is a source differential in the enlarged
source, not a closed source class in the original scalar-reduced branch.

## Proof

The scalar cocycle is non-exact in the reduced source. If
\(\bar c_M=d_{\mathrm{CE}}\eta\), then
\[
  1=\bar c_M(z_1,z_2)
  =
  (d_{\mathrm{CE}}\eta)(z_1,z_2)
  =
  -\eta(\{z_1,z_2\}_{\bar A})
  =
  -\eta(0)=0.
\]
Thus \([\bar c_M]\neq0\), and
\(\hbar N[\bar c_M]\neq0\) for \(N>0\).

The scalar-symbol lift obstruction is exactly this class. The manuscript's
Hom obstruction lemma says a filtered scalar-contact chain projection exists
only when the successive \(o_\sigma^{(r)}\) classes vanish. The first
ordinary-branch evaluation is \(\hbar N\omega(f,g)\gamma_{\mathbf 1}\), so
at \((f,g)=(z_1,z_2)\) it is nonzero.

Assume a scalar projection is later supplied. The fixed-window scalar
closed-exchange equation is solvable only if the class
\(\hbar N[\bar c_M]\) lies in the negative image of
\[
  H^1(\widehat\sigma_{\mathrm{sc},M}\Xi^{\mathrm{Cost},\mathrm{sc}}_M).
\]
For the current genuine tower,
\(\mathcal X^{\mathrm{reg}}\subset\ker\widehat\sigma_{\mathrm{sc},M}\), so
the scalar image is zero. The explicit finite-row data have
\[
  S_{\alpha_{\mathrm{sc}},M}=S_{\alpha_{11},M}=S_{\theta_3,M}=0.
\]
The algebraic cone has the right scalar value only because it adjoins that
value. Distributional contact candidates do not satisfy the current
Costello-local support theorem. Hence the cokernel class is nonzero in the
declared habitat.
