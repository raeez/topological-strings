## Stable core

The minimal full-equivariant finite-window non-scalar QME branch is proved
all orders only in its named habitat.  The habitat is the local
\(\mathbb R^2_{\mathrm{top}}\times\widehat{\mathbb C^2}_{0,\mathrm{hol}}\)
full-equivariant marked branch with \(V=\mathbb C^{N|N}\), generated by the
strict current-interface rows, the one-cross scalar-contact row
\(\alpha_{\mathrm{sc}}\), and its subset-deletion closure, with no added
positive-order seed graph, CE-source face, graph-counterterm face, or nonzero
scalar-zero row.

In that habitat,
\[
  R^{\mathrm{ns}}_{n,M}=0,\qquad
  \mathfrak o^{\mathrm{ns}}_{n,M}=0,\qquad
  C_{n,M}=0
\]
for every \(n\geq 1\).  This is a proved finite-window theorem, not a
statement about unrestricted Costello graph packages.

For any larger package, the scalar-zero condition only permits formation of
the kernel class.  The package must still supply graph type, finite-window
labels, marker tensor, incidence signs, row values, scalar gates, and
truncation coefficients before the \(H^1/\varprojlim^1\) non-scalar class is
defined.
