## Verdict

The finite window admits a precise split escape ledger, but not a finite
\(L_\infty\) transfer retaining \(\mathcal P_{\leq N}\) as surviving
Matlis classes.

The positive algebraic object is the block decomposition of the full
Matlis coadjoint action relative to
\[
  \mathcal P
    =
  \mathcal P_{\leq N}\oplus K_N,\qquad
  K_N=\bigoplus_{a+b>N}\C\rho_{a,b}.
\]
It has four coefficient operations:
\[
  a_f:\mathcal P_{\leq N}\to\mathcal P_{\leq N},\qquad
  b_f:\mathcal P_{\leq N}\to K_N,
\]
\[
  \kappa_f:K_N\to\mathcal P_{\leq N},\qquad
  c_f:K_N\to K_N.
\]
Here \(a_f\) is the projected finite-window action, \(b_f\) is escape out
of the window, \(\kappa_f\) is return to the window, and \(c_f\) is the
high-mode action.  The first forced return is exactly
\[
  \kappa_{z_1^{N+2}}(\rho_{N+1,0})
    =
  -(N+2)\rho_{0,1}.
\]

This package is not finite: closure under \(c_{z_2}\) forces the infinite
chain \(\rho_{N+1,0},\rho_{N+2,0},\ldots\).  If one tries to convert the
return maps \(\kappa_f\) into \(L_\infty\) boundaries while keeping only a
finite window, then every retained principal part becomes exact.  Hence the
exact next obstruction is the escape-return Jacobi class together with the
cohomology-survival obstruction
\[
  \operatorname{im}\kappa_N=\mathcal P_{\leq N}.
\]

## Setup

Fix \(N\geq1\).  Let
\[
  W_N=\mathcal P_{\leq N},\qquad
  K_N=\ker(\pi_N:\mathcal P\to W_N)
      =\bigoplus_{a+b>N}\C\rho_{a,b}.
\]
Let \(A_f\) denote the Matlis coadjoint action of
\(f\in\mathfrak h\) on \(\mathcal P\).  In monomial coordinates,
Appendix `appendix-matlis-principal-parts.tex:163-180` gives
\[
  A_{z_1^pz_2^q}\rho_{i,j}
    =
  -(pj-qi+p-q)\rho_{i-p+1,j-q+1},
\]
with negative-index targets equal to zero.

Relative to \(\mathcal P=W_N\oplus K_N\), write
\[
  A_f=
  \begin{pmatrix}
    a_f & \kappa_f\\
    b_f & c_f
  \end{pmatrix},
\]
where
\[
  a_f=\pi_NA_f|_{W_N},\qquad
  b_f=(1-\pi_N)A_f|_{W_N},
\]
\[
  \kappa_f=\pi_NA_f|_{K_N},\qquad
  c_f=(1-\pi_N)A_f|_{K_N}.
\]
The projected finite-window binary operation on the cotangent side is
\(a_f\).  The first escape-return operation requested in the prompt is
\(\kappa_f\).

## First Forced Escape Row

Two elementary rows decide the first Jacobi attack:
\[
  b_{z_2}(\rho_{N,0})=(N+1)\rho_{N+1,0},
\]
and
\[
  \kappa_{z_1^{N+2}}(\rho_{N+1,0})
    =-(N+2)\rho_{0,1}.
\]
The second identity is the requested forced operation.  The first identity
explains where the killed input \(\rho_{N+1,0}\) comes from inside a
finite projected Jacobiator: it is the escaped intermediate generated by
the finite input \(\rho_{N,0}\) under the Hamiltonian \(z_2\).

Let \(f=z_1^{N+2}\), \(g=z_2\), and \(\rho=\rho_{N,0}\).  The projected
representation Jacobiator is
\[
  J_N(f,g;\rho)
    =
  a_fa_g\rho-a_ga_f\rho-a_{\{f,g\}}\rho.
\]
Since \(a_g\rho=0\), \(a_f\rho=0\), and
\[
  \{z_1^{N+2},z_2\}=(N+2)z_1^{N+1},
\]
one obtains
\[
  J_N(z_1^{N+2},z_2;\rho_{N,0})
    =
  (N+1)(N+2)\rho_{0,1}\neq0.
\]
Equivalently,
\[
  J_N(z_1^{N+2},z_2;\rho_{N,0})
    =
  -\kappa_{z_1^{N+2}}b_{z_2}(\rho_{N,0}).
\]
Thus the forced \(\kappa_N\)-row does account for the first Jacobi failure,
but only after the escape-out map \(b_N\) is also retained.

## Block Identities

The full coadjoint action is a representation:
\[
  [A_f,A_g]=A_{\{f,g\}}.
\]
Splitting this identity into \(W_N\) and \(K_N\) blocks gives the exact
coherence equations:
\[
  [a_f,a_g]-a_{\{f,g\}}
    =
  -\kappa_fb_g+\kappa_gb_f,
\]
\[
  a_f\kappa_g-a_g\kappa_f+\kappa_fc_g-\kappa_gc_f
    =
  \kappa_{\{f,g\}},
\]
\[
  b_fa_g-b_ga_f+c_fb_g-c_gb_f
    =
  b_{\{f,g\}},
\]
\[
  b_f\kappa_g-b_g\kappa_f+[c_f,c_g]
    =
  c_{\{f,g\}}.
\]
These identities are TeX-ready construction data for a split full-current
escape ledger.  They are not a finite \(L_\infty\) structure on
\(\mathcal P_{\leq N}\) alone.

The first equation says precisely that the projected finite-window action
\(a_f\) fails Jacobi by an escape-return term.  The second, third, and
fourth equations say that \(\kappa_N\) is not an isolated correction:
coherence forces \(b_N\) and the high-mode action \(c_N\).

## Obstruction To A Finite \(L_\infty\) Repair

Assume one wants an \(L_\infty\) algebra whose degree-zero visible
coefficient space contains \(\mathfrak h\oplus W_N[1]\), whose binary
operation on visible generators is the projected bracket, and whose unary
differential does not kill the retained Matlis classes in \(W_N\).  The
\(n=3\) \(L_\infty\) identity forces the projected Jacobiator to be an
\(\ell_1\)-boundary:
\[
  J_N(f,g;\rho)=\ell_1\ell_3(f,g,\rho).
\]
For the triple above this gives
\[
  (N+1)(N+2)\rho_{0,1}\in\operatorname{im}\ell_1.
\]
Thus \(\rho_{0,1}\) cannot survive as a finite-window Matlis class.

This is not a one-class accident.  The return map \(\kappa_N\) hits every
retained principal part.  For \(0<r+s\leq N\), take
\[
  f=z_1^{N+1}z_2,\qquad
  k_{r,s}=\rho_{N+r,s}\in K_N.
\]
Then
\[
  \kappa_f(k_{r,s})
    =
  -\bigl((N+1)s-r\bigr)\rho_{r,s}.
\]
The coefficient is nonzero: if \(s=0\), then \(r>0\); if \(s\geq1\), then
\((N+1)s-r\geq1\).  Hence
\[
  \operatorname{im}\kappa_N=W_N.
\]
Any semistrict repair that absorbs the return maps as unary boundaries
therefore makes all retained finite Matlis moments exact.  Such a repair no
longer represents the finite moment output \(p_N\).

## No Finite Escape Color

One might instead keep escape generators rather than killing them by
\(\ell_1\).  This also fails as a finite package.  The first forced row
requires \(\rho_{N+1,0}\) in the escape color.  The high-mode action then
forces
\[
  c_{z_2}(\rho_{m,0})=(m+1)\rho_{m+1,0},
  \qquad m\geq N+1.
\]
Thus any escape color stable under the full Hamiltonian action contains
\[
  \rho_{N+1,0},\rho_{N+2,0},\rho_{N+3,0},\ldots .
\]
It is infinite.  Cutting this chain at any finite height breaks the
second, third, or fourth block identity above and hence breaks Jacobi at
the next window boundary.

## Factorization Compatibility Attack

At the purely algebraic interval-current level, the split block package is
compatible with the already proved current operations.  Tensor
\(W_N\) and \(K_N\) with \(\mathcal D'_c(I)\).  The coefficient maps
\(a_f,b_f,\kappa_f,c_f\) commute with extension by zero, and multiplication
of an interval current by a test function is associative by
`appendix-factorization-current-conventions.tex:190-216`.

This does not give a support-local quotient.  It gives a two-color split of
the full current module:
\[
  \mathcal D'_c(I)\widehat\otimes\mathcal P
    =
  \bigl(\mathcal D'_c(I)\widehat\otimes W_N\bigr)
  \oplus
  \bigl(\mathcal D'_c(I)\widehat\otimes K_N\bigr).
\]
The high color \(K_N\) is not a relation subspace because
\(\kappa_N(\mathfrak h,K_N)=W_N\).  Declaring it to be killed recovers the
strict quotient obstruction; retaining it recovers the infinite full
Matlis tail.

The BMK collar/factorization problem remains separate.  The block
identities are coefficient identities.  They do not prove descent of
\(H_\chi\), do not remove the analytic-versus-finite moment gap, and do
not solve the annular diagonal orientation.  They only identify the exact
algebraic shape of the escape operations forced after the strict quotient
fails.

## TeX-Ready Statement

```tex
\begin{prop}[Projected finite-window escape block and Jacobi obstruction]
\label{prop:bmk-projected-window-escape-block}
Fix \(N\geq1\) and write
\[
  \mathcal P=\mathcal P_{\leq N}\oplus K_N,\qquad
  K_N=\bigoplus_{a+b>N}\C\rho_{a,b}.
\]
For \(f\in\mathfrak h\), let \(A_f\) be the Matlis coadjoint action and
write
\[
  A_f=
  \begin{pmatrix}
    a_f & \kappa_f\\
    b_f & c_f
  \end{pmatrix}
\]
relative to this splitting.  Then
\[
  [a_f,a_g]-a_{\{f,g\}}
    =
  -\kappa_fb_g+\kappa_gb_f,
\]
with the three companion block identities
\[
  a_f\kappa_g-a_g\kappa_f+\kappa_fc_g-\kappa_gc_f
    =
  \kappa_{\{f,g\}},
\]
\[
  b_fa_g-b_ga_f+c_fb_g-c_gb_f
    =
  b_{\{f,g\}},
\]
\[
  b_f\kappa_g-b_g\kappa_f+[c_f,c_g]
    =
  c_{\{f,g\}}.
\]
The first forced return operation is
\[
  \kappa_{z_1^{N+2}}(\rho_{N+1,0})=-(N+2)\rho_{0,1}.
\]
Moreover
\[
  b_{z_2}(\rho_{N,0})=(N+1)\rho_{N+1,0},
\]
so the projected Jacobiator satisfies
\[
  a_{z_1^{N+2}}a_{z_2}\rho_{N,0}
  -a_{z_2}a_{z_1^{N+2}}\rho_{N,0}
  -a_{\{z_1^{N+2},z_2\}}\rho_{N,0}
  =
  (N+1)(N+2)\rho_{0,1}.
\]
Consequently the projected finite-window binary operation is not a Lie
operation on \(\mathcal P_{\leq N}\).  Any \(L_\infty\) repair whose
\(\ell_1\ell_3\) kills this Jacobiator makes \(\rho_{0,1}\) exact.  In
fact \(\operatorname{im}\kappa_N=\mathcal P_{\leq N}\), so such a repair
makes every retained principal part exact.  A finite \(L_\infty\) package
retaining \(\mathcal P_{\leq N}\) as surviving finite moments therefore
does not exist.  The honest escape package is the split full-current
package retaining \(K_N\) and the maps \(b_N,\kappa_N,c_N\).
\end{prop}
```
