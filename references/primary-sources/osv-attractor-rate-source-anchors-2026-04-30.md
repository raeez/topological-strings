## Exact Convention Bridge Required

Before OSV can be compared numerically with CDGP/GV/HKQ/frontier data,
the following bridge must be proved or ledgered explicitly.

1. Fix the OSV Kahler gauge and charge polarisation:
   `C X^Lambda=p^Lambda+i phi^Lambda/pi`, with magnetic charges fixed
   and electric charges summed. This is a mixed ensemble, not a
   microcanonical fixed-curve-class count.
2. Identify OSV's flat topological variable
   `t^A=(p^A+i phi^A/pi)/(p^0+i phi^0/pi)` with the CDGP/HKQ mirror
   coordinate. The proof must say whether the frontier uses the raw
   logarithmic period `omega_1/omega_0` or the normalised flat coordinate
   `(2 pi i)^{-1} omega_1/omega_0`, and must fix branch and sign.
3. Relate OSV `g_top=+-4 pi i/(p^0+i phi^0/pi)` to the GV expansion
   variable `lambda`, including any Wick rotation and the `2 pi` in the
   GV exponential convention.
4. Relate the frontier exponent `exp[-m beta dot t]` to GV's
   `exp[-2 pi k sum_i n_i t_i]`, with class labels, cover variables,
   and the `2 pi` convention kept visible.
5. Derive `z_c=5^{-5}` only from the CDGP mirror-quintic coordinate
   convention `z=(5 psi)^{-5}`. The number `5 log 5=-log z_c` is then a
   CDGP-coordinate arithmetic value, not an OSV source line.
6. Match HKQ table variables `(d,g,n_d^g)`, the KKV/PT-GW/GV
   normalisation, and the conifold flat coordinate to the same `t`.
   HKQ supplies finite BPS table and boundary data, not an OSV rate.
7. Supply a saddle/Laplace-inversion theorem if the OSV mixed
   black-hole partition function is to imply fixed-class quintic
   coefficient growth for `n_d^0`. Without that theorem, OSV remains
   physical ensemble evidence only.
